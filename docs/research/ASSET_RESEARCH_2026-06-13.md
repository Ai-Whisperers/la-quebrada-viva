# Asset library expansion research — 2026-06-13

Push the LQV asset library past 200 unique CC0 / CC-BY-4.0 IDs on disk.

Baseline (2026-06-12): 132 IDs on disk = 10 HDRIs + 42 texture dirs + 80 model blends.
Target: ≥ 200 unique IDs. Delta planned: +200 from Poly Haven (curated) +85 from
ambientCG. Resume-safe download — only the missing items are pulled.

All assets selected here are blanket CC0 (Poly Haven blanket per
<https://polyhaven.com/license>, ambientCG blanket per
<https://docs.ambientcg.com/license>). No CC-BY-SA, no CC-BY-NC. License stubs
are auto-generated by the downloader scripts into `LICENSES/<id>.txt` so the
attribution bundle stays offline-complete.

## Poly Haven additions

The Poly Haven asset catalog was browsed via `https://api.polyhaven.com/assets?t=<type>`;
every slug below was validated to exist in the live catalog response before
landing here. The polyhaven manifest is in `scripts/download_polyhaven_assets.py`
in three new lists `EXTRA_HDRIS`, `EXTRA_TEXTURES`, `EXTRA_MODELS`.

## License verification

- **Poly Haven**: blanket CC0 1.0 Universal per <https://polyhaven.com/license>.
  No per-asset license check required; the license is project-wide. Pulled
  date 2026-06-13 stamped in every `LICENSES/<slug>.txt` stub.
- **ambientCG**: blanket CC0 1.0 Universal per <https://docs.ambientcg.com/license>.
  Same treatment.

Both sites are explicitly allow-listed for the LQV redistribution bundle
(`LICENSE_BUNDLE.md` §1). No new license-text files need to land in `LICENSES/`
— `CC0-1.0.txt` already covers both sources.

## Sources NOT used (and why)

- **Sketchfab** — per-asset license; CC-BY-SA and CC-BY-NC contaminate the
  search results. Bulk pulling unsafe. Existing Sketchfab assets in the repo
  were manually license-cleared one-at-a-time; that workflow doesn't scale to
  +80 picks in a session.
- **OpenGameArt** — same per-asset license problem.
- **Quixel Megascans** — license excludes commercial redistribution to
  third-party clients (Wesley would be a third-party here).
- **TextureCan / 3DTextures.me / others** — licenses unverified case-by-case;
  not blanket-safe.
